    3 Números¶

    Introducción¶

    Como hemos visto hasta ahora, en Javascript tiene dos tipos de números primitivos, el tipo number y el tipo bigInt para números más grandes. El object wrapper para cada uno de ellos es, Number y BigInt, respectivamente.

    Los números pueden ser escritos de forma seguida como 1000 o usando el carácter _ como separador de miles 1_000. En la práctica, no hay diferencia de escribir un número de una forma u otra. Sin embargo, al usar _ como separador hace que el código sea más legible.

    También podemos escribir la sintaxis NeX donde N es el número y donde X es el número de 0 que contiene. También es una forma de expresar la exponentiation científica:

    let billion = 1e9 // 1.000.000.000 ó 1 x 10⁹

let mcs = 1e-6 // 0.000001 ó 1 x 10⁻⁶

    Un número puede ser expresado también en diferente formas según su base:

    • Binarios. Se usa el prefijo 0b. Por ejemplo, `0b11111111' que representa 255 en base decimal.
    • Octal. Se usa el prefijo 0o. Por ejemplo, 0o377, que representa 255.
    • Hexadecimal. Se usa el prefijo 0x. Por ejemplo, 0xff.

    Método toString¶

    El método toString([base]) se encarga de transformar el número a una cadena según la base indica (en caso de no indicarse ninguna base, se obtiene la cadena en base decimal). La base es un número comprendido entre 2 y 36.

    const number = 3

console.log(number.toString()) // "3"
console.log(number.toString(2)) // "11"

    Warning

    Cuando se utiliza un método de números con un literal numérico, es necesario hacer uso de doble puntuación para la llamada del método .., ya que Javascript entiende con un único punto . un valor decimal. En caso de usarlo, daría en Syntax Error

    console.log(1..toString())

    La clase Math¶

    La clase Math es una clase de utilidad (solo almacena funciones y constantes) que contiene varias funciones para poder trabajar con números, así como ciertas constantes. Entre las constantes podemos encontrar:

    • Math.PI: Valor de variable pi.
    • Math.E: Valor del número de Euler.

    Entre los métodos más comunes encontramos:

    • Math.pow(a, b). Calcula la potencia, siendo a la base y b el exponente.
    • Math.sign(number). Comprueba si el número indicado es positivo (devuelve 1), negativo (devuelve -1) o 0 (devuelve -1).
    • Math.sqrt(number). Calcula la raíz cuadrada del número indicado.
    • Math.abs(number). Retorna el valor absoluto de un número.
    • Math.sin(radians). Retorna el seno de un ángulo expresado en radianes.
    • Math.cos(radians). Retorna el coseno de un ángulo expresado en radianes.
    • Math.min(values). Retorna el valor mínimo de una serie de valores.
    • Math.max(values). Retorna el valor máximo de una serie de valores.

    Info

    Descubre más métodos de la clase Math: https://developer.mozilla.org/es/docs/Web/JavaScript/Reference/Global_Objects/Math

    Redondeo¶

    La clase Math contiene, además, una serie de métodos para poder redondear números.

    • Math.floor(number). Redondea hacia abajo, por lo que el número 3.1 se convierte en 3 y -1.1 se convierte en -2.
    • Math.ceil(number). Redondea hacia arriba, así el número 3.1 se convierte en 4 y -1.1 se convierte en -1.
    • Math.round(number). Redondea hacia el valor más cercano, de esa manera 3.1 redondea a 3 y 3.6 redondea a 4.
    • Math.trunc(number). Elimina la parte decimal del número, quedándose con la parte entera. Así, el número 3.6 es 3, y -1.1 es 1.

    Estos métodos sirven para redondear hasta la parte entera, no sirven para redondear en posiciones decimales. Para poder redondear a una décima específica podemos seguir dos formas:

    • Multiplicando y dividiendo. Para redondear un número a una décima n específica, se multiplica el número por 1 seguido de tantos ceros como décimas se quiere redondear. Una vez redondeado, se vuelve a dividir por el mismo número. Por ejemplo, si se desea redondear un número decimal a dos décimas, se multiplicaría y dividiría por 100:

      const number = 1.23456

const mult = number * 100 // 123.456
const round = Math.round(mult) // 123
const div = round / 100 // 1.23

console.log(Math.round(number * 100) / 100) // 1.23

    • Haciendo uso del método toFixed(n) del object wrapper Number, siendo n el número de décimas a mostrar (si no se usa redondea a la parte entera). Es similar al método Math.round(). El problema de este método, es que no solo redondea el número, si no que retorna dicho número como una cadena. Por lo que habría que usar alguna forma de conversión vista en el Tema 2 para poder ser usada como número:

      let num = 12.35

console.log(+num.toFixed(1)) // 12.4

    Comprobación¶

    El método isFinite convierte su argumento a número y comprueba si dicho número es un número regular, es decir, no es Infinity, ni -Infinity, ni NaN.

    console.log(isFinite("15")) // true
console.log(isFinite("str")) // false // (1)!
console.log(isFinite(Infinity)) // false
    1. La conversión de str a número da como resultado NaN.

    El método isNaN convierte su argumento a número y comprueba si el resultado es NaN:

    console.log(isNaN(NaN)) // true
console.log(isNaN("str")) // true

    Warning

    El valor NaN es el único que al comparar con cualquier otro valor, siempre devuelve false, incluyéndose a él mismo. Por eso, no se podría hacer lo siguiente:

    if(NaN === NaN) {} // false

    Número aleatorio¶

    La función Math.random() retorna un número aleatorio comprendido entre 0 y 1. Si se desea obtener un número mayor, se recomienda multiplicar el valor por el límite deseado. Por ejemplo, si se desea obtener un número aleatorio entre 0 y 10, se recomienda multiplicar por 10.

    console.log(Math.random())
console.log(Math.random() * 2)
console.log(Math.random() * 10)
console.log(Math.random() * 100)

    Estos métodos devuelve un número decimal, si se desea obtener un número entero, se puede usar los métodos de redondeo vistos en los apartados anteriores.

    En ciertas ocasiones se desea obtener un valor comprendido entre dos valores, como por ejemplo, entre 10 y 20. Para ello se puede multiplicar el número aleatorio multiplicado por la resta entre el límite máximo y mínimo añadido a una unidad, posteriormente sumándole el valor mínimo al resultado. La formula quedaría así:

    Math.random() * (max - min + 1) + min

    Incluso, se puede obviar los decimales:

    Math.floor(Math.random() * (max - min + 1) +  min)

    Veamos un ejemplo:

    const randomNumberBetween10And20 = Math.floor(Math.random() * 11 + 10) // 20 - 10 + 1

    Ejercicios¶

    Ejercicio 1

    Crea un programa Javascript que solicite varios números, hasta que cancele la petición. Luego muestra, el número más pequeño y el número más grande. Crea dos versiones, una sin usar los métodos Math.min y Math.max creando tus propios métodos, y otra haciendo uso de dichas funciones. Compara los resultados

    Ejercicio 2

    Crea un programa Javascript que solicite al usuario una figura entre las siguientes: Cuadrado, Triángulo, Rectángulo, Círculo. Al elegir una de ellas, se le solicitará, además, los datos necesarios para calcular el área de la figura deseada. Muestra en una alerta, el área redondeada en dos decimales. Si el usuario introduce una figura no válida, se le volverá a solicitar hasta que introduzca una figura válida. Además, realiza las comprobaciones pertinentes.

    Para estos ejercicios se va a crear un fichero llamado numbers.js. En ellos, se va a crear las diferentes funciones indicadas a continuación. Posteriormente, se va a crear otro script para realizar las pruebas pertinentes a mano alzada.

    Method 1

    Crea una función que que dado dos parámetros, devuelva un número aleatorio entre ambos.

    Method 2

    Crea una función que reciba un número y devuelva el número redondeado con el número de décimas indicada. Si recibe un cero, redondea a la parte entera. El método puede no recibir ningún parámetro, en dicho caso el valor del número ha recibir sería 0.

    Method 3

    Crea una función que reciba un número y devuelva el número truncado con el número de décimas indicada. Si recibe un cero, trunca a la parte entera. El método puede no recibir ningún parámetro, en dicho caso el valor del número ha recibir sería 0.

    Method 4

    ¿Sabías que un raíz cuadrada se puede expresar como una potencia? Por ejemplo, la raíz cuadrada de 4, se puede expresar como 4½. Al igual que la raíz cuadrada, cualquier raíz puede ser expresada como potencia, de esta forma, la raíz cuarta de 16, se puede expresar como 16¼. Sabiendo esto, crea un método que reciba dos números, la base y el tipo de raíz a calcular, y haz que devuelva la raíz enésima a calcular.
