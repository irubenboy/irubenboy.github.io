    4 Cadenas¶

    Introducción¶

    Como ya hemos visto en el tema 2 una cadena es un conjunto de caracteres que se pueden formar de tres maneras:

    • Comillas simples.
    • Comillas dobles.
    • Comillas invertidas (backticks)

    Además, las comillas invertidas nos permitían evaluar expresiones haciendo uso de la estructura ${}:

    console.log(`El resultado de 2 + 2 es ${2 + 2}`)

    Otra de las ventajas que tiene este tipo de comillas, es que te permite crear cadenas de multiple líneas, cosa que con el otro tipo de cadenas se debe usar el carácter \n:

    const stringMultiline = `Esto es un cadena
multi línea`

const stringError = "Esto no es una cadena multi línea 
y además da error"

const string = "Esto es una cadena multi línea\nsin errores usando el carácter \\n"

    Caracteres especiales¶

    Existe una serie de caracteres especiales que al escribirlas en una cadena, mostrará algo diferente entre ellas, podemos encontrar:

    Carácter Descripción
    \n Salto de línea
    \r En Windows, los archivos de texto usan una combinación de dos caracteres \r\n para representar un corte de línea, mientras que en otros SO es simplemente '\n'. Esto es por razones históricas, la mayoría del software para Windows también reconoce '\n'.
    \' \" \` Comillas
    \\ Barra invertida
    \t Tabulación
    \b, \f, \v Retroceso, avance de formulario, tabulación vertical – Se mencionan para ser exhaustivos. Vienen de muy viejos tiempos y no se usan actualmente.

    Veamos un ejemplo:

    console.log("Esto es una cadena que usa:\n\t- Comillas simples: \'\n\t- Comillas dobles:\"\n\t- Comillas invertidas: \`\n\t- Barra invertida:\\") //(1)!
    1. SALIDA:
      Esto es una cadena que usa:
      - Comillas simples: '
      - Comillas dobles: "
      - Comillas invertidas: `
      - Barra invertida: \ */

    Si se desea usar la forma literal de algunos de estos caracteres será necesario hacer uso de un doble uso de la barra invertida \\. Veamos un ejemplo:

    console.log(`
En Javascript, para hacer un salto de línea es necesario usar \\n. Mientras que,
para hacer una tabulación es necesario hacer uso del \\t, al igual que con las comillas (\\' \\")
`)

    Tamaño de string¶

    El tipo string tiene diferentes métodos y propiedades que nos permite tratar con las cadenas de una forma más sencilla. Entre ellas, encontramos la propiedad length que permite obtener el tamaño de la cadena.

    const string = 'Hello world'
const stringLength = string.length // 11

    Acceder a caracteres¶

    Para poder acceder a una carácter específico de una cadena, es necesario conocer su posición. Debemos tener en cuenta, que la posición dentro de la cadena de caracteres empieza en 0, por lo que el primer carácter tendrá una posición 0. Al ser la primera posición 0, la última posición será un valor inferior al tamaño de la cadena str.length - 1.

    Para acceder a un carácter de una cadena conociendo su posición, podemos usar corchetes [pos] o el método at(pos):

    const string = "Hello World"

console.log(string[0]) //H

console.log(string[string.length - 1]) // d

console.log(string[2]) // l

console.log(string[11]) // undefined

console.log(string[12]) // undefined

    Una de las gran diferencias al usar corchetes o el método at, es que si a los corchetes se le indica un número negativo, siempre devuelve un undefined. Por el contrario, el método at() con una posición negativa devuelve el carácter empezando a leer la cadena a la inversa:

    console.log(string[-1]) // undefined

console.log(string.at(-1)) // d

console.log(string.at(-11)) //H

console.log(string.at(-12)) // undefined

    String inmutables

    Las cadenas son INMUTABLES por lo que no se puede reasignar un nuevo valor a sus caracteres. Para cambiar el valor de un carácter, será necesario crear una nueva cadena.

    const string = "Hello World"

string[0] = 'A' // ❌❌ERROR

const newString =  'A' + string[1] + string[2] + string[3] //...

    Capitalización¶

    Los métodos para trabajar con la capitalización de cadenas son:

    • toLowerCase(): Transforma la cadena en minúsculas.
    • toUpperCase(): Transforma la cadena en mayúsculas.
    const string = "Hello World"

console.log(string.toUpperCase()) // HELLO WORLD
console.log(string.toLowerCase()) // hello world

    Localizar caracteres en cadenas¶

    Existen diversos métodos para poder comprobar y localizar caracteres dentro de una cadena, como indexOf, lastIndexOf, includes, startsWith, endsWith.

    indexOf¶

    El método indexOf(c) comprueba si el carácter c se encuentra en la cadena y retorna la primera posición en la que se encuentra. Por el contrario, retornará -1, si el carácter no se encuentra en la cadena. Veamos un ejemplo:

    const string = "Hello world"

console.log(string.indexOf('e')) // 1

    En caso de haber más de una coincidencia, retornará el valor de la primera coincidencia, ya que empieza a realizar la búsqueda desde el primer carácter.

    const string = "Hello world"

console.log(string.indexOf('o')) // 4

    El método indexOf(c, [i]), puede recibir un segundo parámetro que indica a partir de que posición buscar el carácter. De esta manera, podemos localizar la siguientes referencias de un carácter:

    const string = "Hello world"

console.log(string.indexOf('o', 5)) // 7

    También podemos localizar una subcadena dentro de la cadena, retornando la primera posición del primer carácter de la misma:

    const string = "This is a large string"

console.log(string.indexOf("string")) // 16

    En el ejemplo anterior, retornará 16 porque la subcadena string comienza en el carácter 16 de la cadena principal. De esta manera podemos hacer comprobaciones de si una cadena contiene una subcadena:

    const string = "This is a large string"

if(string.indexOf("This") > -1){
    console.log(`La cadena ${string} contiene la cadena "this"`)
}

    ¡OJO!

    En el ejemplo anterior hay que tener mucho cuidado, si hacemos la comprobación si la comparación, ya que si el método retorna 0, quiere decir que la cadena si existe, pero al usarlo en el condicional se convertirá a booleano (recuerda que 0 en booleano es false) por lo que no entraría en el bloque if. Por el contrario, si no encuentra la cadena, el método retorna un -1, que al pasarlo a booleano es true, por lo que si ejecutaría el bloque if:

    const string = "This is a large string"

if(string.indexOf("This")){
    console.log(`La cadena ${string} contiene la cadena "this"`)
}

    lastIndexOf¶

    El método lastIndexOf(c, [pos]) es un método bastante similar a indexOf, con la diferencia de que en este caso, la búsqueda comienza a hacerla desde el final de la cadena, por lo que retorna el último valor que parece en la cadena. Veamos un ejemplo:

    const string = "Hello world"

console.log(string.lastIndexOf("o")) // 7
console.log(string.lastIndexOf("o", 5)) // 4

    includes¶

    El método includes(c) retornará true si c se encuentra en la cadena. Por el contrario retornará false. Puede recibir un segundo argumento para indicar desde que posición realizar la comprobación.

    const string = "Hello world"

console.log(string.includes("Hello")) // true
console.log(string.includes("no")) // false

    startsWith¶

    El método startsWith(c) comprueba si la cadena empieza con los caracteres indicados. Retorna true si comienza con dicho carácter (o caracteres) y false al contrario.

    console.log('Hello world'.startsWith('Hello')) // true

    endsWith¶

    El método endsWith(c) comprueba si la cadena termina con los caracteres indicados. Retorna true si termina con dicho carácter (o caracteres) y false al contrario.

    console.log('Hello world'.endsWith('world')) // true

    Obtener subcadenas¶

    El tipo string nos ofrece varios métodos para obtener una subcadena dada una cadena: substring, substr y slice.

    slice¶

    El método slice(start, [end]) retorna una subcadena desde la posición indicado (start) hasta la posición final (end) (no incluido). Si no se indica posición final, recortará hasta el final de la cadena.

    const string = "Hello world"

console.log(string.slice(0, 5)) // Hello
console.log(string.slice(6)) // world

    Se pueden usar valores negativos, para comenzar a cortar desde el final.:

    const string = "Hello world"

console.log(string.slice(-11, -5)) // Hello

    Si se intercambian las posiciones, es decir, si se indica primero la posición mayor que la posición menor, el método slice retorna una cadena vacía.

    console.log("Hello world".slice(2, 1)) // ""

    substring¶

    El método substring(start, [end]) es bastante similar al método slice, pero en este caso, si se usa una posición mayor y luego una menor, este método lo intercambiará. Sin embargo, no se pueden usar argumentos negativos, ya que son tratados como 0.

    const string = "Hello world"

console.log(string.substring(0, 5)) // Hello
console.log(string.substring(6)) // world
console.log(string.substring(-10, -5)) // ""
console.log(string.substring(-10, 5)) // Hello (similar a (0, 5))
console.log(string.substring(2, 1)) // e

    substr¶

    El método substr(start, [length]) recorta una cadena desde la posición inicial indicada, tantos caracteres como indica el segundo argumento (length), a diferente de los otros métodos que se indicaba la posición final. El argumento del tamaño, no puede ser negativo

    const string = "Hello world"

console.log(string.substr(3, 5)) // lo wo
console.log(string.substr(0, 5)) // Hello
console.log(string.substr(6)) // world
console.log(string.substr(-10, -5)) // ""
console.log(string.substr(-10, 5)) // ello
console.log(string.substr(2, 1)) //  l

    Warning

    Este método ha sido indicado como deprecated en las versiones recientes de Javascript, por lo que no se recomienda su uso.

    Comparación¶

    Para poder comparar cadenas en Javascript se realiza la comparación, carácter a carácter. Cada carácter tiene un representación de codificación (en Javascript las cadenas están representadas por la codificación UTF-16), por lo que para comparar los caracteres, será mayor el que mayor codificación tenga. Veamos un ejemplo:

    console.log('a' > 'Z') // tur

    A pesar, de que la a aparezca antes en el abecedario, su codificación corresponde a 97, mientras que la Z a la 90, por lo que ciertamente a es mayor que Z. Lo mismo puede ocurrir con los acentos.

    Con el método codePointAt(pos) retorna el valor codificado del carácter posicionado en la cadena. Si no se indica posición, retornará el valor codificado del primer carácter:

    console.log('a'.codePointAt(0)) // 97
console.log('Z'.codePointAt()) // 90
console.log('Hello World'.codePointAt(5)) // 32

    Por el contrario, podemos obtener el carácter que representa un código a través del método estático String.fromCodePoint(code):

    console.log(String.fromCodePoint(97)) // a

    El algoritmo correcto para realizar comparaciones de strings es más complejo de lo que parece, debido a que los alfabetos son diferentes para diferentes lenguajes. Una letra que se ve igual en dos alfabetos distintos, pueden tener distintas posiciones. Para ello existe el método localeCompare(string, [locales, options]) que permite comparar cadenas teniendo en cuenta el alfabeto del idioma especificado. Sus argumentos son:

    • string. Cadena para comparar.
    • locales. Idioma al comparar. En caso de ser especificado, obtiene el idioma del entorno.
    • options. Un objeto con las configuraciones previas, como la sensibilidad a la mayúsculas.

    El método, además, retornará:

    • 1 si la primera cadena es mayor que la segunda.
    • -1 si la segunda cadena es mayor que la primera.
    • 0 si ambas son iguales.
    console.log("Hola mundo".localeCompare("Hello world", "es")) // 1

    Ejercicios¶

    Para estos ejercicios se va a crear un fichero llamado strings.js. En ellos, se va a crear las diferentes funciones indicadas a continuación. Posteriormente, se va a crear otro script para realizar las pruebas pertinentes a mano alzada.

    Method 1

    Crea un método llamado capitalize(string, [onlyFirst = true]), que transforme la cadena obtenida como argumento a capitalizada. Una cadena es capitalizada, indicando la primera palabra en mayúsculas. Por ejemplo: 'Hello world'. Recibirá un segundo argumento booleano para comprobar si se hace una capitalización en cada palabra. Por ejemplo, Hello World. En caso de no recibir este parámetro solo capitalizará la primera letra.

    Method 2

    Crea una función truncate(str, maxlength, [ellipsis = '...']) que verifique la longitud de str y, si excede maxlength reemplaza el final de str con el carácter indicado como tercer argumento que por defecto será puntos suspensivos, para hacer su longitud igual a maxlength.

    truncate("Lo que me gustaría contar sobre este tema es:", 20) //"Lo que me gustaría c…"
truncate("Hola a todos!", 20) // "Hola a todos!"
    Method 3

    Crea una función jump(string, maxLength) que inserte saltos de línea cada vez que la cadena llegue a maxLength.

    jump("hola mundo esto es una cadena super larga con muchos saltos de línea", 5)

    Method 4

    Crea dos funciones, isEmpty(string) e isNotEmpty(string), que compruebe si string es una cadena vacía o no, respectivamente.

    Method 5

    Crea dos funciones, isBlank(string) e isNotBlank(string) que comprueba si string es una cadena vacía o con espacios en blanco; o no, respectivamente.

    Method 6

    Realiza un método isPalindrome(string) que comprueba si la cadena string es palíndroma, es decir, se puede leer de igual forma de izquierda a derecha que derecha a izquierda (sin tener en cuenta los espacios ni las comas o puntos.).
