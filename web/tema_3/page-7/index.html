    7 Transformaciones¶

    Introducción¶

    CSS 3 permite realizar diferentes tipos de transformaciones visuales 2D y 3D sobre las cajas, a través de una serie de funciones de transformación y de propiedades. Así, podremos trasladar, escalar, rotar y deformar una caja, tanto en dos dimensiones (ejes x e y), como en tres dimensiones (eje z).

    Las transformaciones NO afectan al flujo normal de posicionamiento de cajas, ya que se llevan a cabo una vez ha concluido el posicionamiento. Tan sólo hacen que los elementos luzcan de forma diferente.

    Propiedades¶

    La propiedad transform permite especificar las transformaciones que deben realizarse sobre un elemento. Recibe como valor una lista de funciones de transformación que deben aplicarse, separadas por espacios en blanco. Las funciones de transformación se realizan en el orden en el que aparecen.

    Propiedad transform
    Valores none |
    Se aplica a Elementos transformables
    Valor inicial none
    Descripción Establece las funciones de transformación que se deben aplicar al elemento

    Veamos unos ejemplos:

    .rotada {
  transform: rotate(-15deg);
}
.rotada-deformada {
  transform: rotate(90deg) skewx(60deg);
}

    Por su parte, la propiedad transform-origin permite modificar las coordenadas del punto de referencia usado como origen de las transformaciones a aplicar a un elemento. Por defecto dicho punto de referencia es el centro del elemento.

    Propiedad transform-origin
    Valores
    Se aplica a Elementos transformables
    Valor inicial 50% 50% 0
    Descripción Establece las coordenadas del punto de referencia origen de las transformaciones en el elemento

    El desplazamiento en cada eje se puede expresar mediante:

    • Una longitud, indicada en alguna de las unidades conocidas.
    • Un porcentaje respecto de la anchura / altura de la caja del elemento.
    • Valores left, right, center, top, bottom.

    Veamos un ejemplo:

    .rotada-left-top {
  transform-origin: left top;
  transform: rotate(90deg);
}

    Funciones de transformación 2D¶

    Traslación¶

    Las funciones de traslación son aquellas que desplazan el elemento en alguno de los ejes X o Y. Los valores positivos desplazan hacia la derecha en el eje X y hacia abajo en el eje Y, mientras que los valores negativos desplazan hacia la izquierda en el eje X y hacia arriba en el eje Y. Si se indican porcentajes, éstos se calculan respecto a la anchura/altura del elemento.

    • translate(x, y): Traslada el elemento una distancia x horizontalmente y una distancia y verticalmente.
    • translateX(x): Traslada el elemento una distancia x horizontalmente.
    • translateY(y): Traslada el elemento una distancia y verticalmente.

    Por ejemplo, transform: translate(10px, -20px) traslada el elemento 10 píxeles a la derecha y 20 píxeles hacia arriba. Es equivalente a transform: translateX(10px) translateY(-20px)

    Escalado¶

    Las funciones de escalado permiten aumentar o reducir el tamaño del elemento. Reciben el factor de escalado, de manera que si es positivo el elemento se agranda y si es negativa se empequeñece.

    • scale(fx, fy): Escala el elemento horizontalmente en un factor fx y verticalmente en un factor fy. Si fx y fy corresponden al mismo valor, se mantendrá la proporción original del elemento al escalarlo.
    • scaleX(fx): Escala el elemento horizontalmente en un factor fx.
    • scaleY(fy): Escala el elemento verticalmente en un factor fy.

    Por ejemplo, transform: scale(3, 2) hace que el elemento tenga el triple de ancho y el doble de alto que originalmente.

    Rotaciones¶

    as funciones de rotación giran el elemento el número de grados especificado, en el sentido de las agujas del reloj.

    • rotate(deg): Rota el elemento respecto al eje Z el número de grados indicados.
    • rotateX(deg): Rota el elemento respecto al eje X el número de grados indicados.
    • rotateY(deg): Rota el elemento respecto al eje Y el número de grados indicados.

    Por ejemplo, transform: rotate(15deg) rota el elemento 15 grados en el eje Z.

    Deformaciones¶

    Las funciones de deformación inclinan el elemento el número de grados indicados.

    • skew(xdeg, ydeg): Inclina el elemento el número de grados indicados en el eje X y el número de grados indicados en el eje Y.
    • skewX(deg): Inclina el elemento en el eje X el número de grados indicados.
    • skewY(ydeg): Inclina el elemento en el eje Y el número de grados indicados.

    Por ejemplo, transform: skew(20deg, 15deg) inclina el elemento 20 grados en el eje X y 15 grados en el eje Y.

    Funciones de transformación 3D¶

    CSS proporciona funciones de transformación similares a las anteriores pero sobre el eje Z, que corresponde al eje de profundidad, lo que nos permitirá modificar el elemento en tres dimensiones. Cada elemento con z > 0 se hace más grande y con z < 0 se vuelve más pequeño.

    Pero para que podamos visualizar correctamente los efectos 3D y determinar su intensidad, es necesario establecer la distancia entre el plano z= 0 y el usuario, es decir, determinar el punto de perspectiva del usuario. Las partes del elemento cuyo valor en el eje Z sean mayores que el valor de perspectiva no serán visibles, ya que quedan "detrás del usuario".

    Para ello tenemos dos opciones:

    • Establecer la propiedad perspective en el elemento padre del elemento al que queremos transformar, asignándole como valor la distancia deseada.
    • Usar la función perspective(distancia) dentro de la propiedad transform del elemento a transformar.

    Un valor bastante adecuado como distancia de perspectiva es 1000px.

    La posición del elemento desde el que se mide la distancia indicada para la propiedad perspective corresponde, por defecto, al centro del elemento, pero podemos modificar dicho punto de referencia mediante la propiedadperspective-origin del elemento padre del elemento a transformar.

    Veamos las funciones de transformación en 3D disponibles:_

    • translateZ(z): Traslada el elemento una distancia z en el eje de profundidad. Los valores positivos acercan el elemento al usuario y los negativos lo alejan.
    • translate3d(x, y, z): Traslada el elemento una distancia x en el eje X, una distancia y en el eje Y y una distancia z en el eje Z.
    • scaleZ(fz): Escala el elemento en el eje Z en un factor de profundidad fz. scale3d(fx, fy, fz): Escala el elemento en el eje X en un factor fx, en el eje Y en un factor fy y en el eje Z en un factor fz.
    • rotateZ(deg): Rota el elemento respecto al eje Z el número de grados indicados. Similar a rotate(deg).
    • rotate3d(x, y, z, deg): Rota el elemento el número de grados indicados, tomando como punto de referencia el denotado por x, y, z.
    • matrix3d(n, n, ...): Establece una matriz de transformación 3D de 16 valores. Se usa en transformaciones mucho más complejas que las anteriores. Visite el siguiente enlace para saber más sobre las matrices de transformación en 3D.
