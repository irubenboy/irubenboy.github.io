    4 Bordes¶

    Introducción¶

    CSS permite modificar el aspecto de cada uno de los cuatro bordes de la caja de un elemento. Para cada borde se puede establecer su anchura o grosor, su color, su estilo, el radio de las esquinas e incluso una imagen para dibujarlo. por lo que en total CSS define 31 propiedades relacionadas con los bordes.

    Anchura¶

    La anchura de los bordes se controla con las cuatro propiedades siguientes:

    Propiedades border-top-width, border-right-width, border-left-width, border-bottom-width
    Valores (unidad de medida | thin | medium | thick ) | inherit
    Se aplica a Todos los elementos
    Valor inicial medium
    Descripción Establece la anchura de cada uno de los cuatros bordes de los elementos

    La anchura de los bordes se indica mediante una medida (en cualquier unidad de medida absoluta o relativa) o mediante las palabras clave thin (delgado), medium (normal) y thick (ancho).

    La unidad de medida más habitual para establecer el grosor de los bordes es el px, ya que es la que permite un control más preciso sobre el grosor. Las palabras clave apenas se utilizan, ya que el estándar CSS no indica explícitamente el grosor al que equivale cada palabra clave, por lo que pueden producirse diferencias visuales entre navegadores.

    Si se quiere establecer de forma simultánea la anchura de todos los bordes de una caja, es necesario utilizar una propiedad shorthand llamada border-width:

    Propiedades border-width
    Valores (unidad de medida | thin | medium | thick ) {1,4} | inherit
    Se aplica a Todos los elementos
    Valor inicial medium
    Descripción Establece la anchura de todos los bordes del elemento

    La propiedad border-width permite indicar entre uno y cuatro valores. El significado de cada caso es el habitual de las propiedades shorthand:

    • Si se indica un solo valor, se aplica a los cuatro bordes.
    • Si se indican dos valores, el primero se aplica al borde superior e inferior y el segundo valor se aplica al borde izquierdo y derecho.
    • Si se indican tres valores, el primero se aplica al borde superior, el segundo se aplica al borde izquierdo y derecho y el tercer valor se aplica al borde inferior. Si se indican los cuatro valores, el orden de aplicación es superior, derecho, inferior e izquierdo.

    Color¶

    El color de los bordes se controla con las cuatro propiedades siguientes:

    Propiedades border-top-color, border-right-color, border-left-color, border-bottom-color
    Valores color | transparent | inherit
    Se aplica a Todos los elementos
    Valor inicial -
    Descripción Establece el color de cada uno de los cuatro bordes de los elementos

    CSS incluye una propiedad shorthand llamada border-color para establecer de forma simultánea el color de todos los bordes de una caja:

    Propiedades border-color
    Valores (color | transparent ) {1,4} | inherit
    Se aplica a Todos los elementos
    Valor inicial -
    Descripción Establece el color de todos los bordes del elemento

    En este caso, al igual que sucede con la propiedad border-width, es posible indicar de uno a cuatro valores y las reglas de aplicación son idénticas a las de la propiedad border-width.

    Estilo¶

    CSS también permite establecer el estilo de cada uno de los bordes mediante las siguientes propiedades:

    Propiedades border-top-style, border-right-style, border-left-style, border-bottom-style
    Valores none | hidden | dotted | dashed | solid | double | groove | ridge | inset | outset | inherit
    Se aplica a Todos los elementos
    Valor inicial none
    Descripción Establece el estilo de cada uno de los cuatro bordes de los elementos

    El estilo de los bordes sólo se puede indicar mediante alguna de las palabras reservadas definidas por CSS, como hidden (oculto), dotted (punteado), dashed (con guiones), solid (línea continúa), double (línea doble), groove (ranura 3D), ridge (marco 3D), inset (hundido hacia dentro), outset (resaltado hacia fuera).

    Como el valor por defecto de esta propiedad es none, los elementos no muestran ningún borde visible a menos que se establezca explícitamente un estilo de borde

    Para establecer de forma simultánea los estilos de todos los bordes de una caja, es necesario utilizar la propiedad shorthand llamada border-style:

    Propiedades border-style
    Valores (none | hidden | dotted | dashed | solid | double | groove | ridge | inset | outset) {1, 4} | inherit
    Se aplica a Todos los elementos
    Valor inicial -
    Descripción Establece el estilo de todos los bordes del elemento

    Como es habitual, la propiedad permite indicar de uno a cuatro valores diferentes y las reglas de aplicación son las habituales de las propiedades shorthand.

    Bordes laterales¶

    Como sucede con los márgenes y los rellenos, CSS define una serie de propiedades de tipo shorthand que permiten establecer los atributos descritos anteriormente a los bordes de forma simultánea. CSS incluye una propiedad shorthand para cada uno de los cuatro bordes y una propiedad shorthand global.

    Propiedades border-top, border-right, border-left,border-bottom
    Valores (grosor | color | estilo) | inherit
    Se aplica a Todos los elementos
    Valor inicial -
    Descripción Establece el estilo completo de cada uno de los cuatro bordes de los elementos

    Veamos un ejemplo:

    h1 {
  border-bottom: solid red;
}

    En el ejemplo anterior, la anchura del borde será la correspondiente al valor por defecto (medium). Este otro ejemplo muestra la forma habitual utilizada para establecer el estilo de cada borde:

    div {
  border-top: 1px solid #369;
  border-bottom: 3px double #369;
}

    Borde global¶

    Por ultimo, CSS define una propiedad de tipo shorthand global para establecer el valor de todos los atributos de todos los bordes de forma directa:

    Propiedades border
    Valores (grosor | color | estilo) | inherit
    Se aplica a Todos los elementos
    Valor inicial -
    Descripción Establece el estilo completo de todos los bordes de los elementos

    Las siguientes reglas CSS son equivalentes:

    div {
  border-top: 1px solid red;
  border-right: 1px solid red;
  border-bottom: 1px solid red;
  border-left: 1px solid red;
}

div { border: 1px solid red; }

    Como el valor por defecto de la propiedad border-style es none, si una propiedad shorthand no establece explícitamente el estilo de un borde, el elemento no muestra ese borde. Por ejemplo:

    /* Sólo se establece el color, por lo que el estilo es
    "none" y el borde no se muestra */
div { border: red; }

    Cuando los cuatro bordes no son idénticos pero sí muy parecidos, se puede utilizar la propiedad border para establecer de forma directa los atributos comunes de todos los bordes y posteriormente especificar para cada uno de los cuatro bordes sus propiedades particulares. Por ejemplo:

    h1 {
  border: solid #000;
  border-top-width: 6px;
  border-left-width: 8px;
}

    Esquinas redondeas¶

    Para hacer que las esquinas de los bordes tengan aspecto redondeado podemos usar las cuatro propiedades siguientes:

    Propiedades border-top-left-radius, border-top-right-radius, border-bottom-left-radius, border-bottom-right-radius
    Valores [|]{1,2}
    Se aplica a Todos los elementos
    Valor inicial 0
    Descripción Establece el radio de curvatura de uno de los cuatro bordes de los elementos

    Para especificar el radio, podemos usar como valor:

    • Una longitud expresada en px, por ejemplo 10px.
    • Un porcentaje, por ejemplo 10%.

    En realidad podemos proporcionar hasta dos valores:

    • Si proporcionamos un único valor, éste corresponderá al radio horizontal y al vertical.
    • Si proporcionamos dos valores, el primero corresponderá al radio horizontal y el segundo al vertical.

    Si queremos especificar de una vez el radio de las cuatro esquinas, podemos usar la propiedad shorthand border-radius:

    Propiedades border-radius
    Valores [ | ]{1,4} [ / [ | ]{1,4} ]?
    Se aplica a Todos los elementos
    Valor inicial 0
    Descripción Establece el radio de curvatura de los bordes de los elementos

    Podemos especificar entre 1 y 8 valores:

    • Si especificamos un único valor, éste corresponderá al radio horizontal y vertical de las cuatro esquinas.
    • Si especificamos más de uno, deben indicarse en el siguiente orden: superior izquierda, superior derecha, inferior derecha e inferior izquierda (siguiendo las agujas del reloj).
    • Si se omite el valor para la inferior izquierda, se le asignará el valor de la superior derecha (el opuesto en la diagonal). Si se omite la inferior derecha, se le asignará el valor de la superior izquierda.t
    • Para establecer diferentes valores para el radio horizontal y el radio vertical de una esquina, usaremos el formato radio_horizontal / radio_vertical.
    • Una aplicación curiosa de esta propiedad es hacer que un elemento sea totalmente redondo. Bastará con que tenga la misma anchura que altura y establezcamos el radio de los cuatro bordes al valor 50%.

    También podemos hacer que el borde sea semitransparente, en cuyo caso se mezclará, por defecto, con el color/imagen de fondo del elemento. Si no queremos que se fusione con el color de fondo del elemento sino con el padre, podemos usar la propiedad background-clip: padding-box para indicar que el fondo del elemento se dibuje detrás del contenido y del padding, pero no del borde.

    Otro efecto curioso es simular que los bordes redondeados son interiores. Para ello, simplemente introduciremos una caja en otra, y haremos que los bordes de la interior queden redondeados.

    Borde formado por imagen¶

    La propiedad border-image permite usar imágenes en los bordes. Es dependiente de la existencia de un área de borde, es decir, que si border-width es 0 la propiedad border-image no se aplicará.

    El área de la imagen del borde, a diferencia del área del borde, se extiende a través de todo el elemento, incluyendo las otras áreas del modelo de caja (border area, padding area, content area). Esto permite utilizar la imagen del borde (a pesar de su nombre) no sólo como fondo del elemento, sino que también permite extender la imagen del borde fuera de la zona del borde.

    La propiedad border-image es la forma acortada de otras propiedades que controlan el uso de imágenes en los bordes:border-image-source, border-image-slice, border-image-outset, border-image-repeat y border-image-width.

    La propiedad border-image-source define la ruta a la imagen que se utilizará como borde. La imagen se mostrará en lugar de cualquier otro valor que tenga border-style declarado. La forma en que se aplica la imagen dependerá de los valores de las otras cuatro propiedades de border-image.

    Propiedades border-image-source
    Valores url | none
    Se aplica a Todos los elementos
    Valor inicial none
    Descripción Establece la imagen a utilizar como borde

    Por ejemplo:

    .module {
  border-image-source: url("border.png");
}

    La propiedad border-image-slice define cuánto se recorta la imagen para crear el borde. Los valores especifican la cantidad de píxeles desde el borde exterior hacia adentro de la imagen que se preservarán al recortar. Debemos tener en cuenta que la cantidad indicada NO controla la anchura del borde, sino tan sólo define la zona a preservar al recortar.

    Propiedades border-image-slice
    Valores [ | ]{1,4} | fill
    Se aplica a Todos los elementos
    Valor inicial none
    Descripción Establece la imagen a utilizar como borde

    El valor fill indica que la imagen NO debe recortarse. En vez del valor fill, puede recibir de 1 a 4 valores:

    • De 1 a 4 valores numéricos: Indican la cantidad en píxeles (NO hay que incluir la unidad px) desde cada borde de la imagen hacia dentro que hay que preservar de la imagen al recortarla (indicados como arriba, derecha, abajo e izquierda).
    • De 1 a 4 porcentajes: Indican el porcentaje del alto/ancho de la imagen desde cada borde hacia adentro que hay que preservar al recortar. En el siguiente ejemplo se preservarán 20 píxeles desde cada borde de la imagen hacia adentro para conformar el borde de la caja:

    Al aplicar una zona de recorte, se crea internamente una cuadrícula de 9 celdas, donde la celda central es recortada y el resto se usará para dibujar las diferentes zonas del borde.

    La propiedad border-image-outset define cuánto se extiende la imagen del borde fuera del área del borde. Puede recibir un valor número, que representa múltiplos del valor de border-width, o un valor en px (incluyendo la unidad px)

    Propiedades border-image-outset
    Valores [ | ]{1,4}
    Se aplica a Todos los elementos
    Valor inicial 0
    Descripción Establece cuánto se extiende la imagen del borden fuera del área de borde

    La propiedad border-image-repeat define cómo se trata la imagen del borde una vez recorta, escalándose o repitiéndola, si no es lo suficientemente grande para cubrir todo el área del borde.

    Propiedades border-image-repeat
    Valores [stretch | repeat | round | space]{1,2}
    Se aplica a Todos los elementos
    Valor inicial stretch
    Descripción Establece el modo de repetición de la imagen de borde dentro del área de borde

    Puede tomar los siguientes valores para los bordes horizontales y verticales (en este orden):

    • stretch: La imagen se estira para rellenar la anchura de borde
    • repeat: La imagen se repite para rellenar la anchura de borde
    • round: La imagen forma un mosaico. Si no se rellena el área con un número entero de repeticiones, cada imagen se reescala.
    • space: La imagen forma un mosaico. Si no se rellena el área con un número entero de repeticiones el espacio sobrante se redistribuye entre cada imagen. No soportado aún por Chrome.

    Note

    NO debemos pensar que a partir de una única imagen completa podemos crear un borde en forma de mosaico repitiéndola por todo el borde. Es necesario crear una nueva imagen que incluya la imagen original en cada una de las 9 celdas de la cuadrícula y después recortarla haciendo uso de border-image-slice.

    La propiedad border-image-width define el ancho real de la imagen del borde. Aunque, por defecto, el ancho es el mismo que el border-width, la imagen del borde puede cubrir más o menos de él.

    Propiedades border-image-width
    Valores [ || | auto ]{1,4}
    Se aplica a Todos los elementos
    Valor inicial 1
    Descripción Define el ancho real de la imagen de borde

    Podemos especificar un valor para los cuatro lados o especificar el valor de cada lado, con las siguientes posibilidades:

    • Un número, que representa múltiplos del valor de border-width
    • Un porcentaje respecto del del área de borde
    • Un valor en píxeles (incluyendo la unidad px)

    Podemos especificar todas las propiedades anterior de una sola vez haciendo uso de la propiedad shorthand border-image.

    Propiedades border-image
    Valores []
    Se aplica a Todos los elementos
    Valor inicial -
    Descripción Establece una imagen de borde

    Sólo los dos primeros valores son obligatorios.
