    4 Pseudoclases¶

    Introducción¶

    CSS introdujo el concepto de pseudoclase para permitir la selección de elementos basándose en información que reside fuera del árbol de etiquetas del documento HTML y que no puede ser expresada usando otro tipo de selector. Por ejemplo, si queremos aplicar un estilo diferencial a los enlaces cuando el puntero del ratón se sitúa sobre él, la información necesaria la selección es cambiante y dependiente del usuario.

    Un selector de pseudoclase está formado por el carácter dos puntos : seguido del nombre de la pseudoclase y, opcionalmente, un valor entre paréntesis. Estos selectores se pueden combinar con cualquier otro selector complejo.

    Veamos los selectores de pseudoclases más importantes, organizados por funcionalidad.

    Pseudoclases para enlaces¶

    Los selectores de pseudoclases relacionados con los enlaces son:

    • :link: Selecciona enlaces que no hayan sido visitados aún.
    • :visited: Selecciona enlaces que ya han sido visitados con anterioridad. Por ejemplo:
    a:link {
  color: green;
}

a:visited {
  color: red;
}

    En este ejemplo los enlaces no se hayan visitado se verán de color verde y los ya visitados de color rojo.

    Pseudoclases de interacción del usuario¶

    Los selectores de pseudoclases relacionados con la interacción del usuario son:

    • :hover: Selecciona elementos que estén siendo apuntados por el usuario mediante un dispositivo apuntador, por ejemplo el puntero del ratón, pero aún no lo ha activado. El ejemplo más característico es cuando el usuario sitúa el puntero del ratón encima de la caja de un elemento.
    • :active: Selecciona elementos que están siendo activados en ese momento por el usuario. El ejemplo más característico es mientras el usuario está haciendo click por el puntero del ratón sobre la caja de un elemento.
    • :focus: Selecciona elementos que en ese momento tengan el foco.

    Por ejemplo:

    a:link {
  color: green;
}
a:visited {
  color: red;
}

a:hover {
  background-color: black;
  color: white;
}

a:active {
  text-decoration: none;
  color: brown;
  font-weight: bold;
}

input[type="text"]:focus {
  background-color: yellow;
}

    En este ejemplo,los enlaces que no se hayan visitado se verán de color verde y los ya visitados de color rojo. Por otra parte, cuando situemos el puntero del ratón por encima de un enlace, su fondo se pondrá de color negro y el texto de color blanco. Además, mientras el usuario esté haciendo click sobre un enlace (y no haya soltado el botón del ratón) el enlace se verá sin subrayado, de color marrón y en negrita. Finalmente, las etiquetas <input> cuyo atributo type tenga el valor text cuando tengan el foco se verán con el fondo de color amarillo.

    Warning

    Si vamos a especificar varios selectores de pseudoclases para los enlaces, se recomienda respetar el orden del ejemplo anterior respecto a :link, :visited, :hover y :active.

    Pseudoclases estructurales¶

    Los selectores de pseudoclases estructurales permiten seleccionar elementos basándonos en su relación con el contenido del padre o de los hermanos:

    • elemento:nth-child(numero): selecciona el elemento indicado pero con la condición de que sea el hijo enésimo de su padre. Este selector es útil para seleccionar el segundo párrafo de un elemento, el quinto elemento de una lista, etc.
    • elemento:nth-last-child(numero): idéntico al anterior pero el número indicado se empieza a contar desde el último hijo.
    • elemento:empty: selecciona el elemento indicado pero con la condición de que no tenga ningún hijo. La condición implica que tampoco puede tener ningún contenido de texto.
    • elemento:first-child y elemento:last-child: seleccionan los elementos indicados pero con la condición de que sean respectivamente los primeros o últimos hijos de su elemento padre.
    • elemento:nth-of-type(numero): selecciona el elemento indicado pero con la condición de que sea el enésimo elemento hermano de ese tipo.
    • elemento:nth-last-of-type(numero), idéntico al anterior pero el número indicado se empieza a contar desde el último hijo.
    • :root: selecciona el elemento raíz del documento. En el caso de HTML, dicho elemento siempre será el elemento <html>.
    • elemento:first-of-type: selecciona el elemento indicado pero con la condición de que sea el primer hijo de ese tipo dentro de su padre. Es similar a elemento:nth-of-type(1).
    • elemento:last-of-type: selecciona el elemento indicado pero con la condición de que sea el último hijo de ese tipo dentro de su padre. Es similar a elemento:nth-last-of-type(1).
    • :only-child: selecciona los elementos que sean hijos únicos, es decir, aquellos que no tienen ningún hermano.
    • elemento:only-of-type: selecciona los elementos que no tengan ningún hermano de dicho tipo.

    Pseudoclases lógicas¶

    La pseudoclase de negación :not() recibe como argumento un selector, y selecciona aquellos elementos que no serían seleccionados por dicho selector. Veamos algunos ejemplos:

    /* Selecciona los li que no sean de la clase diferente */
li:not(.diferente) { color: red; }

/* Selecciona los a que no tengan el foco */
a:not(:focus) { color: red; }

    Note

    La versión CSS4 de esta pseudoclase permite recibir varias condiciones como argumento, de manera que se seleccione aquellos elementos que no cumplan ninguna de las condiciones recibidas. La versión de CSS4 no está soportada por la mayoría de los navegadores.

    CSS4 define otros dos pseudoclases lógicas, cuyo soporte actual por parte de los navegadores es nulo:

    • :matches(): Selecciona los elementos que cumplan alguna de las condiciones pasadas como argumento.
    • :has(): Selecciona los elementos que cumplan con alguno de los selectores pasados como argumento.

    Otras pseudoclases¶

    Otros selectores de pseudoclases son:

    • :lang(idioma): Selecciona los elementos cuyo idioma sea el indicado, sin tener en cuenta el método empleado por el navegador para averiguar el idioma de cada elemento. NO es exactamente igual al selector [lang|=idioma], ya que éste sólo selecciona los elementos para los que explícitamente se ha establecido el atributo lang y su valor contiene el idioma indicado. Por ejemplo, si aplicamos la siguiente regla CSS: p:lang(es) { color: red; }

    • :target: Selecciona el elemento cuyo id corresponde al nombre de sección incluido en la URL que se está visualizando en el navegador. Por ejemplo si se está visualizando la URL http://www.wikipedia.com/index.html#referencias>, el selector de pseudoclase :target() selecciona el elemento cuyo id sea referencias. Se usa, sobre todo, para crear ventanas popup, que se muestran unicamente cuando se pulsa sobre un determinado enlace. Veamos una demostración:

    También existen selectores de pseudoclases relacionados con el CSS para páginas impresas: :blank, :first, :left, :right.

    CSS4 define otra serie de selectores de pseudoclases cuyo soporte actual por parte de los navegadores es prácticamente nulo: :valid-drop, :invalid-drop, :active-drop, :local-link(), :any-link, :past, :current(), :future, :nth-column, :nth-last-column, :nth-match, :nth-last-match, :placeholder-shown, :scope, :blank, :dir(), :fullscreen, :ancestor(), :host, :top.

    Ejercicio

    Crea una página web que contenga 10 enlaces dentro de un div. A dos de ellos le vas a indicar como clase enlace. Aplica los siguientes estilos:

    • A los dos primeros enlaces deben aparecer de normal, sin estar subrayados.
    • Los tres últimos enlaces deben aparecer de color rojo.
    • Los enlaces que sean pares deben tener una fuente distinta.
    • Los enlaces que sean múltiplos de 4 deben aparecer en negrita.
    • Al hacer hover en los enlaces debe aparecer de color rosa.
    • Al activar el enlace debe salir en cursiva.
    • Si el enlace ha sido visitado debe aparecer en naranja.
    • Al hacer hover en el div, cambiará el fondo a cualquier color pero con transparencias.
    • Al hacer hover en los enlaces pares deberá cambiar a color verde.
    • Al activar los enlaces que no tengan la clase enlace deberá aparecer en amarillo.
